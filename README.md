# GameChoice – сервис для коллективного выбора игры

**GameChoice** — это веб-сервис, который позволяет группе людей совместно выбрать игру для вечернего отдыха.  
Пользователь создаёт комнату, добавляет участников и игры, после чего каждый участник голосует за понравившиеся варианты.  
Когда все готовы, сервис случайным образом выбирает игру на основе распределения голосов.

Проект реализован как полноценное клиент-серверное приложение с авторизацией, CRUD-операциями и механизмом онлайн-обновлений через webwebsocket-ы.

## Основные возможности

- **Регистрация и авторизация** (JWT + Refresh)
- **Создание комнат** для голосования
- **Приглашение участников** по имени
- **Добавление и удаление игр** внутри комнаты
- **Голосование за одну или несколько игр**
- **Онлайн-обновление состояния** (добавление игр, голоса, выбор результата)
- **Случайный выбор игры** с учётом процента набранных голосов
- **История результатов** внутри комнаты

## Как это работает

1. Пользователь создаёт новую комнату.
2. Приглашает участников по их уникальному имени.
3. Участники добавляют свои варианты игр.
4. Каждый голосует за понравившиеся игры.
5. Сервис формирует взвешенный рандом на основе количества голосов и выбирает игру.
6. Результат видят все участники в режиме реального времени.

Проект покрывает все базовые требования FullStack-приложения:  
аутентификацию, управление сущностями, ролевые ограничения, работу с БД и реал-тайм обновления.
