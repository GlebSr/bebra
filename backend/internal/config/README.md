# Пакет конфигурации

Пакет `config` предоставляет функционал для загрузки и управления настройками приложения из YAML-файлов.

## Основные возможности

- **Загрузка конфигурации** из YAML-файлов
- **Преобразование конфигурации** в JSON и обратно
- **Типизированный доступ** к настройкам базы данных и сервера
- **Генерация DSN** для подключения к PostgreSQL
