# Пакет Middleware (Промежуточное ПО)

Пакет `middleware` содержит промежуточное ПО (middleware) для HTTP сервера, которое обрабатывает запросы на различных этапах их жизненного цикла.

## Назначение

### Context Middleware
**Назначение:** Добавление уникального идентификатора запроса и подробное логирование всего жизненного цикла запроса.
