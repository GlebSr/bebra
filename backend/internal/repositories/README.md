# Пакет Repository (Репоэитории данных)

Пакет `repository` содержит реализации доступа к данным, инкапсулируя работу с различными источниками данных (БД, внешние API, файлы и т.д.).

## Назначение

Репоэитории предоставляют абстракцию над источниками данных:
- **Изоляция бизнес-логики** от деталей хранения данных
- **Единый интерфейс** для работы с разными источниками
- **Транзакционность** и управление соединениями

## Структура репоэитория для PostgreSQL

### Обязательная структура для каждого репоэитория, связанного с Postgres:
repositories/
├── your_repo/ # Репоэиторий
│ ├── makefile # Сборка и кодогенерация (Брать из примера)
│ ├── queries/ # SQL запросы
│ │ ├── users.sql # SQL файлы с запросами (Брать из примера + гугла)
│ │ └── ...
│ └── gen/ # СГЕНЕРИРОВАННЫЙ код (НЕ РЕДАКТИРОВАТЬ!)
│ └── repository.go # Нами описанное повидение
└── ...
