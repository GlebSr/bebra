# Пакет Handler (Обработчики HTTP запросов)

Пакет `handler` содержит HTTP обработчики (хэндлеры) для API endpoints сервера.

## Назначение

Хэндлеры отвечают за:
- **Прием HTTP запросов** от клиентов
- **Вызов бизнес-логики** через сервисный слой
- **Формирование HTTP ответов** (статусы, данные, ошибки)
